# 音乐节拍图片播放器

一款支持音乐节拍同步的图片播放器，适用于安卓端个人使用。

## 功能特性

- ✅ **自动播放图片**：支持从相册文件夹选择图片并自动播放
- ✅ **音乐BPM分析**：自动分析音乐节奏（BPM），使图片切换频率与音乐节奏保持一致
- ✅ **内置节拍器**：支持自定义BPM、多种节拍声音、非等速节拍
- ✅ **同步控制**：图片切换与音乐/节拍器节拍完全同步
- ✅ **多种转场效果**：弹跳、放大弹出、旋转进入、淡入淡出、滑动、缩放等
- ✅ **全屏播放**：支持全屏模式，横竖屏自动适配
- ✅ **播放模式**：支持顺序播放和随机播放

## 快速开始

### 在安卓手机上使用

**最简单的方法：**

1. **双击运行 `start-server.bat`**（Windows）或 `start-server.sh`（Mac/Linux）
2. **查看显示的IP地址**（例如：192.168.1.100）
3. **在手机浏览器中输入**：`http://192.168.1.100:8000`
4. **确保手机和电脑在同一WiFi网络**

详细说明请查看 [deploy-guide.md](deploy-guide.md)

## 使用方法

1. **选择图片**：点击"选择相册文件夹"按钮，选择包含图片的文件夹
2. **选择模式**：
   - **音乐模式**：选择音乐文件，系统会自动分析BPM并同步图片切换
   - **节拍器模式**：使用内置节拍器，可自定义BPM和节拍声音
3. **设置转场效果**：选择随机转场或自定义转场效果
4. **开始播放**：点击"播放"按钮开始同步播放

## 技术说明

- 纯前端实现，使用原生 HTML/CSS/JavaScript
- 使用 Web Audio API 进行音频处理和BPM分析
- 使用 CSS3 动画实现转场效果
- 支持 PWA（渐进式Web应用），可添加到主屏幕
- 支持 Android Chrome 浏览器的文件夹选择功能

## 浏览器兼容性

- Android Chrome（推荐）
- 其他现代浏览器（部分功能可能受限）

## 注意事项

- 首次使用需要用户授权访问文件和音频
- 建议在 HTTPS 环境下使用以获得最佳体验
- 大量图片时请注意内存使用
- 手机和电脑需在同一WiFi网络才能通过局域网访问
